大陆关押最久台谍刑满释放 曾参与“少康专案”计划

2019-01-29 04:00 赵觉珵

【环球时报记者 赵觉珵】据台湾中时电子报28日报道，因参与代号为“少康专案”的计划，于1999年被大陆逮捕的台湾间谍杨铭中近日刑满获释，23日回到台湾。依照惯例，杨铭中之后将向台湾“国防部军事情报局”作“任务归询”报告。台媒称，服刑19年的杨铭中是台湾被大陆关押最久的情报人员。

上世纪90年代，台湾情报部门策划“少康专案”计划，策反大陆少将刘连昆与大校邵正忠，导致大陆军方大量情报被泄露到台湾。台湾媒体表示，1996年“台海危机”后，大陆方面从台湾时任领导人李登辉的相关讲话中，发觉可能存在情报泄露问题。经调查，大陆于1999年将刘连昆与邵正忠二人逮捕，担任联络人的台谍杨铭中和姚嘉珍也相继被捕。

据台湾《中国时报》报道，杨铭中于2000年被判死刑，后改死缓2年，再改无期徒刑，又改为20年，加上获减刑数月，于2019年1月刑满获释。去年年中，与杨铭中一同被捕的台谍姚嘉珍已经刑满获释。本月23日，杨铭中在家人陪同下自北京搭机返台，台湾“军情局”派人在桃园国际机场迎接。依照惯例，杨铭中还将向“军情局”作“任务归询”报告。

早在2009年，台湾前“军情局”少将处长陈虎门就曾呼吁大陆释放杨铭中等台湾情报人员。2015年11月，台湾“国防部”宣布，两岸此前进行被俘情报员的交换，2006年被捕的台湾“军情局”上校朱恭训与徐昌国获释，但杨铭中却不在其中。

台湾东森新闻云28日称，台湾情报人员在大陆被捕共有两波高峰期：一是1996年“台海危机”时期，被捕人员以军职人员居多;二是2003年，台湾前领导人陈水扁在竞选中公开大陆沿岸部署的导弹数量与位置，引起大陆方面警觉，导致潜伏的30名台湾间谍被捕，其中至少17人被判处10年以上刑期。
